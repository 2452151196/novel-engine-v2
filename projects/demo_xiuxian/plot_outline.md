# 剧情大纲

楚询追查灭门凶手 → 发现蚀骨粉与万宝神殿有关 → 与林药儿结盟 → 深入锈带 → 揭开商盟天下与幽暗联邦的幕后交易 → 以力破局，反杀幕后主使

## 分章节拍

### 第1章：霜鸣示警
- HOOK: 霜鸣剑在鞘中自结霜——危险先至，楚询不动
- 林药儿分析魔瘴成分，楚询只给三秒判断：可战/退/死
- 暗市李三试探，楚询三招拆穿所有伪装，但不杀——留着他还有用
- 结尾：书架灰尘遇剑生青光，有东西在下面动

### 第2章：暗市规矩
- 商盟天下的铜钱规矩：情报即商品，人命有标价
- 林药儿被商会盯上，因为她认出了蚀骨粉成分
- 楚询以力破规：不按商会定价买情报，直接拿人质换
- 打脸：商会管事以为吃定散修，结果霜鸣剑架在脖子上才想起楚询的战绩

### 第3章：锈带入口
- 深入锈带，魔气浓度让普通修士无法呼吸
- 林药儿的药理知识成救命稻草，楚询的剑成开路工具
